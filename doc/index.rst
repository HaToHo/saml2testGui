Saml2testGui
============

:Release: |release|
:Date: |today|

Saml2testGui is a web based interface used which in combination with the SAML2test tool will allow an independent
validation of a specific instance of a SAML2 entity. It will test not only if the instance works according to the
SAML2 standard but also if it complies with a specific profile of SAML2.
